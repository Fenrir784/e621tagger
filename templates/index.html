How to use e621tagger

1. 📤 Upload an image
2. 🏷️ Add/remove tags before copying
3. 💽 Copy tags in needed format

🏷️ Tag interaction
• Purple – high confidence (≥ Confident threshold).
• Blue – medium confidence (≥ All threshold).
• Gray – low confidence (below thresholds).
• Click a tag to force it to be included (green) or excluded (red) from copying.
• Long press or right-click a tag to show the description from e621 wiki.

💄 Pro Tip: Narrow down basic detection so that it is almost always correct, and then manually add or remove tags.

📋 Copying tags
• Use Copy Confident Tags or Copy All Tags buttons to copy all tags that meet the threshold.
• Within each category, small C (Confident) and A (All) buttons copy only tags from that category.

⚙️ Settings
• Theme: System, Light or Dark – follow your preference.
• Thresholds: Choose a preset or set custom values.
• Max tags: Limit the number of returned tags from backend.
• Default format: e621 (space-separated) or PostyBirb (comma-separated).

💡 Extra features
• PWA: Install the app via the browser's "Install" button for quick access.
• Privacy: All processing is done locally on the server and no data is sent to third parties.
